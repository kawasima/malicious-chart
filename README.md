# malicious-chart

インタラクティブな3D円グラフのデモアプリケーションです。スライダーで傾きや回転をリアルタイムに操作できます。

**Demo:** https://kawasima.github.io/malicious-chart/

## 技術スタック

- React 19 + TypeScript 5
- Highcharts (3D Pie Chart)
- Bulma (CSS)
- Vite

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`docs/` に GitHub Pages 用の静的ファイルが出力されます。
